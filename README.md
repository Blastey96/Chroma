Chroma | Professional Painting Suite
Chroma is a custom, web-based digital painting application built to streamline the artistic workflow through a minimalist interface and powerful, specialized tools.
🎨 About the Project
Chroma began as a personal project born out of a desire for a painting environment tailored specifically to my own artistic preferences. While professional suites offer vast arrays of features, Chroma focuses on the specific mechanical and aesthetic needs I value most in my digital art practice: a clean UI, highly responsive brush dynamics, and intuitive shortcuts.
This application is designed to run directly in the browser, providing a portable and powerful painting experience without the overhead of traditional desktop software.
🚀 Features
•	Advanced Layer System: Supports blending modes, alpha locking, clipping masks, and full layer masking.
•	Custom Brush Engine: A high-performance dab-based engine with adjustable flow, hardness, spacing, and scatter.
•	Reference Navigator: A floating, draggable reference window that allows for live canvas monitoring or external image importing with built-in color sampling.
•	Precision Selection: A robust Lasso tool with additive (Shift) and subtractive (Alt) modes and unified visualization.
•	Professional Filters: Real-time Noise generation and Chromatic Aberration effects for post-processing.
•	Multi-Document Support: Work on multiple canvases simultaneously with a specialized tab system.
🛠️ Built With
Chroma was developed using a modern web stack:
•	HTML5 Canvas API for high-performance rendering.
•	Tailwind CSS for a responsive, dark-mode optimized UI.
•	JavaScript (Vanilla) for core application logic and pixel manipulation.
AI Assistance & Inspiration
This project was developed with the assistance of Google Gemini, which helped in architecting the core brush engine, implementing complex raster logic for selections, and refining the user interface.
The functionality and design philosophy of Chroma were heavily influenced by industry-standard painting programs, most notably Krita, with a focus on bringing that level of professional control to a lightweight web environment.
